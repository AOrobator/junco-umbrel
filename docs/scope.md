# Scope

## Product intent
Mobile-first, native-feeling Umbrel app that exposes Sparrow Server capabilities with a delightful UX.

## Design principles
- Thumb-first layout, excellent one-handed use
- Minimal chrome, high contrast, legible in bright light
- Progressive disclosure for advanced options
- Clear error states and recovery paths
- Expose information only when it is needed and actionable

## MVP flows
- Onboarding
- Create/open wallet
- Connect to node (Umbrel Bitcoin + Electrs)
- Receive
- Send
- PSBT import/export
- Labels and notes

## Out of scope (initial)
- Full parity with Sparrow Desktop power features
- Theme customization
- Multi-window workflows
